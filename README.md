### FEATURED — ARES

**Adaptive AI mock-interview engine.** Simulates real technical interviews end-to-end — not a scripted quiz, a system that reacts.

| Layer | What it does |
|---|---|
| **LangGraph** | State-machine core — adaptive depth scaling, skill rotation, early-stop logic |
| **LangFuse** | Full observability — centralized trace wrapper on every LLM call |
| **ChromaDB** | Semantic dedup — cosine-similarity matching kills repeat questions live |
| **Supabase** | Shareable scorecards — UUID-keyed results, one link, zero login |
| **Whisper + Groq/LLaMA** | Voice-driven interview loop, low-latency inference |
